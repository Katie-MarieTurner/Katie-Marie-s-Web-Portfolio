Owl Database

A collection of data on over 240 species of owl.


This project was created using:

  • mySQL

I created this project due to my fascination with owls.

Combined, the database holds the species classification, scientific & common name, current population trend, conservation status, generation length and movement patterns on over 240 species.

Using primary and foreign keys, the tables are linked and thus the user is able to recall specific data, such as the common name along with the conservation status without the unrequested data.
