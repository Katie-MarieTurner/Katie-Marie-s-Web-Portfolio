To-Do List

Manage tasks, mark as completed and remove tasks.


This project was created using:

  • HTML & CSS
  • JavaScript
  • VSCode

This project is a minimalistic to-do list, which can:

  • Add & remove custom tasks
  • Mark tasks as completed and change the style accordingly
  • Prompt the user to fill the task box if add task (or Enter) is pressed

I would like to develop a way of prioritising tasks by complete date.
