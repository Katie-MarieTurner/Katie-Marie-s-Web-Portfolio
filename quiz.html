Owl Quiz

A multi-question quiz that tracks the user's score.


This project was created using:

  • Python & tkinter
  • VSCode

This project was started when my interest in creating GUI's
for my python projects began.

The project is a collection of multiple choice questions which takes in the user's answers, validates them, stores an accumulative total and moves onto the next question automatically.

In future, I would like to improve the GUI design as my knowledge and use of tkinter improves with practice.
