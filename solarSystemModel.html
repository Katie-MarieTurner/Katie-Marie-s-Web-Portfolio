Solar System Model

A model solar system using relative sizing.


This project was created using:

  • HTML & CSS
  • JavaScript
  • VSCode

I developed this project to stretch my ability in CSS. Finding solutions to varied orbit times and relative planet sizing was challenging but also a lot of fun.

It is a basic model solar system, with relative orbits & sizes.

One of the inaccuracies is the starting points of the planets' orbits, I would like to fix this in future. I would also love to add more detail to the style of the model, and mabe in future add moons!
